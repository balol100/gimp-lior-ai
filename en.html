What is PhotoGIMP?

GIMP is a free, powerful image editor — but its interface is different from Photoshop and takes time to get used to. That's where PhotoGIMP comes in: it's a small add-on that rearranges the tools, shortcuts and look of GIMP so it feels exactly like Adobe Photoshop.

Frequently asked questions

Is PhotoGIMP really free?
Yes, completely. Both GIMP and PhotoGIMP are open-source, entirely free software — no subscription, no hidden cost and no time limit.
Do I need to install GIMP too?
Yes. PhotoGIMP is an add-on that sits on top of GIMP. First install GIMP 3.0 from the official site, then add the PhotoGIMP add-on following the guide above.
Does it work in my language?
Absolutely. GIMP supports interfaces in many languages, including right-to-left (RTL) text. You can choose your language via Edit → Preferences → Interface → Language.
Are the keyboard shortcuts the same as Photoshop?
Yes — that's the whole point. PhotoGIMP maps the keyboard shortcuts to Adobe's official documentation so you feel at home from the first moment. You can always change a shortcut via Edit → Keyboard Shortcuts.
Will it delete my brushes and fonts?
No. PhotoGIMP only replaces configuration files (shortcuts, layout, preferences). Your personal brushes, fonts and plugins stay untouched.
How do I uninstall it?
Simply delete GIMP's configuration folder (3.0) and reopen it — it will generate clean default settings. If you made a backup earlier, you can restore it.
